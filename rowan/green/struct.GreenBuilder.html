pub struct GreenBuilder { /* fields omitted */ }

Construction cache for green tree elements.

As the green tree is immutable, identical nodes can be deduplicated. For example, all nodes representing the #[inline] attribute can be deduplicated and refer to the same green node in memory, despite their distribution throughout the source code.
